# custom_label
カスタム可能なタスク・メモアプリを作るべく、奮闘するプロジェクト。

dockerインストール後、
docker-up.sh を実行して、
コンテナを作成してください。

その後、コンテナ内で以下を実行します。
```
./yarn_install.sh
```

インストールが完了したら、
yarn serveでサーバを起動してください。

http://localhost:8080/
